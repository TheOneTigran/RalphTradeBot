# 🤖 RalphTradeBot

**RalphTradeBot** — это продвинутая гибридная аналитическая система для торговли криптовалютами, объединяющая строгую математическую обработку рынка с логикой искусственного интеллекта (LLM). Система специализируется на **Волновом Анализе Эллиотта (Elliott Wave Theory)** с использованием фрактальной геометрии и адаптивной оценки волатильности.

---

## 🧠 Как работает алгоритм (AI-Wave Engine V2)

Пайплайн включает в себя 5 последовательных этапов:

### 1. Сбор данных (On-Chain & Orderbook Mapping)
Бот подключается к потокам данных биржи (по умолчанию Bybit V5) и собирает:
* Исторические свечи (OHLCV) сразу по **шести таймфреймам** (от `1W` до `5m`).
* **Open Interest (OI) & Funding Rates:** Для определения истинной силы тренда (набираются ли шорты или лонги, назревают ли ликвидации).
* **Глубокий стакан (Bid/Ask Walls):** Бот выявляет сверхкрупные лимитные ордера покупателей и продавцов, которые могут служить магнитом для цены (Smart Money).

### 2. Математическое Ядро (Adaptive ATR-Fractal)
Вместо классического ZigZag с фиксированным процентом отклонения или обычных индикаторов, RalphTradeBot применяет собственный динамический алгоритм:
* **Фракталы Вильямса** находят все потенциальные микроэкстремумы на графике.
* **ATR Фильтр** (с индивидуальными настройками для каждого таймфрейма) отфильтровывает рыночный шум. Маленькие движения на младших таймфреймах игнорируются, формируя чистые, непрерывные направленные векторы (свинги).
* **Фибо-Кластеры:** Мощный инструмент, ищущий математические совпадения (Confluences). Алгоритм строит коррекции и расширения (0.382, 0.618, 1.618 и т.д.) каждого найденного свинга. Там, где уровни с разных таймфреймов накладываются друг на друга (с мини-погрешностью 0.5%), образуется «Кластер». Чем больше совпадений, тем выше показатель `strength` (Сила).

### 3. Actor LLM (Чтение структуры рынка)
Сжатый, предварительно обработанный математический JSON передается в систему Искусственного Интеллекта.
ИИ, руководствуясь строгим сводом правил (89WAVES), читает структуру и:
* Оценивает дивергенции RSI и Awesome Oscillator.
* Взвешивает кластеры Фибоначчи.
* Строит волновые проекции и дает точную разметку на всех таймфреймах сразу (фрактально).
* Генерирует основной и альтернативный сценарий с выделением процента уверенности (Multi-Count).

### 4. Self-Correction (Actor-Critic Pipeline)
Самая важная фишка V2. Нейросети свойственно галлюцинировать и нарушать строгие математические правила.
1. Actor отправляет черновик своего торгового плана аудитору (LLM Critic).
2. Critic проверяет черновик по железным законам: "зашла ли 4 волна за 1?", "является ли 3 волна самой короткой?".
3. Если найдено структурное противоречие, Critic возвращает ошибку, и Actor переписывает план, исправляя маркировку.

### 5. Визуализация и форматирование (Trading Plan)
Когда план верифицирован, бот:
* Парсит его в структурированную базу данных.
* Генерирует красивые читабельные текстовые отчеты.
* С помощью `mplfinance` генерирует графики (PNG) с отображением вычисленных векторов (зигзагов) и сильных Фибо-кластеров.

## 🛠 Установка

### 1. Подготовка окружения
Клонируйте репозиторий и установите библиотеки Python:
```bash
pip install -r requirements.txt
```

### 2. Настройка (`.env`)
Скопируйте `.env.example` в `.env` и впишите туда ваши ключи:
```ini
# Bybit Keys
BYBIT_API_KEY="ВАШ_КЛЮЧ"
BYBIT_API_SECRET="ВАШ_СЕКРЕТ"

# OpenRouter / OpenAI (Для работы ИИ)
OPENAI_API_KEY="ВАШ_КЛЮЧ"

# Telegram Bot Token (От @BotFather)
TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"

# Настройки по умолчанию
SYMBOLS="BTCUSDT"
TIMEFRAMES="1d,4h,1h,15m,5m"
```

## 💻 Команды и Использование

### 1. Telegram Бот (Рекомендуемый способ)
Запустите Telegram-сервер:
```bash
python telegram_bot.py
```
> Бот работает асинхронно, может обрабатывать несколько запросов одновременно и не зависнет при ожидании ответа от ИИ.

**Доступные команды в боте:**
* `/start` — Приветствие и справка.
* `/analyze <ТИКЕР> [ТАЙМФРЕЙМЫ]` — Старт полного пайплайна с графиком и верификацией.
  * *Пример 1:* `/analyze ETHUSDT` (Глубокий анализ со стандартными ТФ)
  * *Пример 2:* `/analyze DOGEUSDT 1W,1d,4h` (Анализ по выбранным ТФ)

Ответом бот пришлёт:
1. Моноширинный HTML текстовый отчёт (Торговый план).
2. Набор PNG графиков с подписями уровней. 

### 2. Терминал (CLI)
Для запуска скрипта локально на вашем компьютере:
```bash
python main.py
```
*(По умолчанию проанализирует монеты из `.env`)*

**Флаги CLI:**
* `--symbols` — Тикеры через пробел.
* `--timeframes` — Таймфреймы через пробел.
* `--no-critic` — Пропустить Self-Correction проверку в целях экономии LLM-токенов / времени.
* `--no-oi` — Не запрашивать данные открытого интереса.

**Пример использования из терминала:**
```bash
python main.py --symbols BNBUSDT SOLUSDT --timeframes 1d 4h 1h 15m --no-critic
```

---
*Все отчеты и графики, сгенерированные локально через `main.py` или через бота, сохраняются в директорию `/reports` с отметкой времени.*
